Tutorial

How to use

Here is a step-for-step guide on how to use Covered:

  1. Create a Covered Account:
    1. Enter all your User Credentials
    2. If you have problems finding your Steam ID follow this Tutorial: Ubisoft Support on how to find your Steam Id
    3. Make sure, that your Steam Profile is public: Steam Support on how to make your Profile Public
  2. You should be automaticly logged in
  3. Now you have to select the Covers you like.
  4. The 4 Buttons:
    • "Save": You set this Cover as your Cover for this Game and you wont get any recommendations for that Game anymore (you should be carefull with this option)
    • "Like": You add this Cover to your Liked Covers. When your finished you can select your Favorite Cover from those Covers
    • "Dislike": You just skip this Cover
    • "Dislike Game": You wont get any Covers for this Game anymore
  5. Finishing
    1. Select your Favorite Covers out of your Selection
      • Go to the "Profile" Page
      • Select the Game you want to select your Cover for
      • Now press the "Save" button on the bottom right of the Cover you like the most
      • Do this for all of your Games
    2. Download your Folder through the "Download" page
    3. Copy all the Files from your Downloaded ZIP file and paste them into your Steam LibraryCache folder ('C:\Program Files (x86)\Steam\appcache\librarycache')

About this this Project

Covered is a WebApp, with the intention to make it as easy as possible to customize your Steam Library with your Favorite Covers. Therefore it uses the Steam API to access your Game Library and give you recommendations on which games you could be interested in. Then it goes to search the Covered Database to look for Covers for those games. If it doesnt find any Covers it will try to find some Covers through the SteamGridDB API. It will then present you the Covers it found and you can select, which one you like the best. As soon as you'r finished with this process, you can Download a ZIP file, containing all Game Covers. You only have to paste those files into the Steam LibraryCache Folder.
